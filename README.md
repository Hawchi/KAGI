# KAGI

Educatieve website over custom mechanische keyboards, gebouwd voor beginners die hun eerste build willen builden.

KAGI legt uit hoe een custom keyboard in elkaar zit, welke onderdelen je nodig hebt en hoe je 'm zelf bouwt. Inclusief een interactieve build guide, een verkenner voor onderdelen en een 3D keyboard op de homepage.

## Features

- Interactieve onderdelen verkenner (case, PCB, plate, switches, stabilizers, keycaps, foam en mods)
- 3D keyboard model op de homepage (Three.js)
- Switch geluid preview met de Web Audio API

## Stack

- Vue.js
- Three.js
- Web Audio API
